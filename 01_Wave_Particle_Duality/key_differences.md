# Key Differences: Wave-Particle Duality

This table summarizes the fundamental conflict between the classical and quantum views regarding the nature of objects.

| Feature | Classical View | Quantum View |
| :--- | :--- | :--- |
| **Identity** | An object is either a particle OR a wave. | An object (e.g., an electron) is a single entity that exhibits **both** particle-like and wave-like properties. |
| **Localization** | Particles are localized at a single point. Waves are delocalized. | A quantum object travels as a delocalized wave but arrives (is detected) as a localized particle. |
| **Behavior** | Particles collide. Waves interfere. | A single quantum object can interfere with itself, demonstrating its wave nature. |
| **Double-Slit** | A particle must go through one slit or the other. | The quantum object goes through **both** slits simultaneously as a wave. |
| **Analogy** | Billiard Balls vs. Water Ripples | A "Probability Wave" that collapses into a particle upon measurement. |
